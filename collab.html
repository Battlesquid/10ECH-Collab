Job Opportunities

Position 1: Looking for Quality Assurance Engineers, Software Engineers, and Firmware Engineers who can either debug software code, maintain code, and write code.
Position 2:High demand for Firmware Engineers that can remove viruses that are disrupting workflow. Software Engineers are also welcome to debug broken code that may be causing the issue.

Skills that we are looking for include -
  • Source Control, preferably experience with the Git command line.
  • Debugging code. Software Engineers and Firmware Engineers are most applicable in this. Knowing languages like C or C++ also helpful.
  • Good maintenance skills. Being able to revise products for quality is also an advantage.
  • Teamwork skills are also valued. Must be able to cooperate and complete tasks efficiently.
  • Certification (eg, a bachelors degree in your field).
